# ydtechpro

A B2B/B2C hybrid marketplace platform for IT and Security infrastructure.

## Features
- **Buyer View**: Post leads, browse products, and negotiate.
- **Seller View**: Manage inventory, submit quotes, and track negotiations.
- **Admin Dashboard**: Identity verification, lead registry, and inventory auditing.
- **Real-time Negotiation**: WebSocket-based chat for price negotiations.
- **Secure Payments**: Integrated with Razorpay.

## Tech Stack
- **Frontend**: React, TypeScript, Tailwind CSS, Framer Motion.
- **Backend**: Node.js, Express, Socket.io.
- **Database**: MySQL (Aiven.io) with JSON fallback.
- **Deployment**: Render.com.

## Setup
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Configure environment variables in `.env`.
4. Run development server: `npm run dev`.
